# TodoList Vue 3 練習專案

## 設置專案
```
npm install
```

### 開啟內部 API
```
npm run api
```

> 重要！下載後請務必輸入此指令，會開啟一個內部的 json-server 後端程式 ( 監聽 port http://localhost:3000 )，您可以訪問 `http://localhost:3000/todos` 取得 todos 的資料。

### 開發模式
```
npm run serve
```

### 部署模式
```
npm run build
```


### Lints and fixes files
```
npm run lint
```
